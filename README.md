# Password-Analyzer-Hash-Cracker-Demo
Образовательный инструмент на Python для анализа силы паролей, генерации хэшей и демонстрации уязвимостей слабых паролей.

**Важное предупреждение**  
Этот проект создан исключительно в образовательных целях.  
НЕ используйте его для взлома чужих аккаунтов или несанкционированного доступа — это незаконно и нарушает этику кибербезопасности.

### Что умеет проект
- Оценка силы пароля по критериям OWASP (длина, разнообразие символов, распространённые шаблоны)
- Генерация хэшей: MD5, SHA-256, SHA-512, bcrypt (рекомендуемый алгоритм)
- Демонстрация dictionary-атаки на слабые пароли (только маленький встроенный словарь для демо)
- Безопасный ввод пароля (без отображения на экране)
- Подчёркнута важность современных алгоритмов хэширования и многофакторной аутентификации

### Технологии
- Python 3
- Стандартные библиотеки: hashlib, getpass, re, time
- bcrypt (опционально, для безопасного хэширования — `pip install bcrypt`)
